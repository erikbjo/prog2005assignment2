alive Boolean (optional) a boolean to filter alived characters

Errors

The Countries Dashboard Service uses standard HTTP status codes to indicate the success or failure of the API call. In general: Codes in the 2xx range indicate success. Codes in the 4xx range indicate an error that failed given the information provided (e.g., a required parameter was omitted, a charge failed, etc.). Codes in the 5xx range indicate an error with Countries Dashboard's servers (these are rare).
